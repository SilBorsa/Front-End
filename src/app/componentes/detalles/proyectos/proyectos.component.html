Sistema de Ajustes Contables - (1999)

Fué mi primer gran proyecto, con el que más aprendí. Era un sistema en el que cada usuario habilitado cargaba las modificaciones Contables que consideraba necesarias para el análisis a su cargo, luego un usuario de nivel superior lo autorizaba y finalmente se imprimía para cargarlo formalmente en el sistema propio. Realizado con MS Access 97 y Visual Basic. Se accedía con usuario y clave, tenía un menú para navegar entre las ocpiones disponibles, según los niveles de acceso. Había formularios de carga, formularios de consulta y reportes con diferentes tipos de impresiones (detallada, sumarizada, por cuenta contable, por organismo, etc). Estuvo vigente por 3 años, hasta que se diseñó el sistema nuevo (SIDIF gráfico en ese momento) y se incorporó la idea de los ajustes.
